145   # Declare that the mock object should receive a message with the given name.
 
146   #
 
147   # If more than one method name is given, then the mock object should expect
 
148   # to receive all the listed melthods.  If a hash of method name/value pairs
 
149   # is given, then the each method will return the associated result.  Any
 
150   # expectations applied to the result of +should_receive+ will be applied to
 
151   # all the methods defined in the argument list.
 
152   #
 
153   # An expectation object for the method name is returned as the result of
 
154   # this method.  Further expectation constraints can be added by chaining to
 
155   # the result.
 
156   #
 
157   # See Expectation for a list of declarators that can be used.
 
158   #
 
159   def should_receive(*args)
 
160     @last_expectation = ContainerHelper.parse_should_args(self, args) do |sym|

162       result = Expectation.new(self, sym)
 
163       @expectations[sym] << result
 
164       override_existing_method(sym) if flexmock_respond_to?(sym)
 
165       result
 
166     end
